**Γλωσσάρι Όρων Αγροτικών Ενισχύσεων**

**1. Γενικοί Όροι & Διαδικασίες**

| Τίτλος                                      | Κατηγορία / Είδος                | Σημασία (απλά)                                                                                                                               |
| :------------------------------------------- | :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------- |
| **Check List (Λίστα Ελέγχου)**             | Διαδικασία Επεξεργασίας         | Έντυπο με βασικά σημεία ελέγχου και παρατηρήσεις για κάθε αίτηση. Επιστρέφεται μετά την επεξεργασία.                                       |
| **ΟΣΔΕ (Ολοκληρωμένο Σύστημα Διαχείρισης και Ελέγχου)** | Σύστημα Πληρωμών                 | Το ηλεκτρονικό σύστημα μέσω του οποίου υποβάλλονται οι αιτήσεις και διαχειρίζονται οι πληρωμές των αγροτικών ενισχύσεων.                      |
| **Αίτηση Ενιαίας Ενίσχυσης (ΑΕΕ)**          | Είδος Αίτησης                   | Η κύρια αίτηση που υποβάλλουν οι παραγωγοί για να λάβουν τις άμεσες ενισχύσεις.                                                              |
| **Πύλες Εισόδου**                          | Σύστημα Υποβολής                | Οι φορείς (π.χ. ΚΥΔ - Κέντρα Υποδοχής Δηλώσεων) μέσω των οποίων οι παραγωγοί μπορούν να υποβάλουν την ΑΕΕ.                                  |
| **GDPR (Γενικός Κανονισμός Προστασίας Δεδομένων)** | Νομικό Πλαίσιο                  | Κανονισμός για την προστασία των προσωπικών δεδομένων των παραγωγών.                                                                         |
| **Α/Α (Αύξων Αριθμός)**                     | Ταυτοποίηση                     | Μοναδικός αριθμός που δίνεται σε κάθε αγροτεμάχιο μέσα στην αίτηση ενός παραγωγού.                                                              |
| **Scanner (Φάκελος Δικτύου)**              | Αποθήκευση Εγγράφων             | Κοινόχρηστος φάκελος στο δίκτυο όπου αποθηκεύονται τα σκαναρισμένα δικαιολογητικά των παραγωγών.                                              |
| **Ομαδοποίηση Αιτήσεων**                   | Διαδικασία Επεξεργασίας         | Περίπτωση όπου αιτήσεις διαφορετικών παραγωγών (συνήθως συγγενών) συνδέονται και επεξεργάζονται μαζί.                                          |
| **Πολλαπλή Συμμόρφωση (Cross Compliance)** | Υποχρεώσεις Παραγωγού          | Κανόνες που πρέπει να τηρούν οι παραγωγοί σχετικά με το περιβάλλον, τη δημόσια υγεία, την υγεία των ζώων και την καλή γεωργική κατάσταση.         |
| **ΚΑΠ (Κοινή Αγροτική Πολιτική - CAP)**     | Ευρωπαϊκή Πολιτική              | Η πολιτική της Ευρωπαϊκής Ένωσης για τη γεωργία, η οποία χρηματοδοτεί τις ενισχύσεις.                                                        |
| **Πρώτος Πυλώνας (ΚΑΠ)**                    | Χρηματοδότηση Ενισχύσεων        | Περιλαμβάνει τις άμεσες ενισχύσεις προς τους γεωργούς (π.χ. βασική ενίσχυση, συνδεδεμένες).                                                  |
| **Δεύτερος Πυλώνας (ΚΑΠ)**                  | Χρηματοδότηση Ενισχύσεων        | Περιλαμβάνει μέτρα αγροτικής ανάπτυξης (π.χ. βιολογικά, εξισωτική).                                                                          |

**2. Είδη Ενισχύσεων & Προγραμμάτων**

| Τίτλος                                                | Κατηγορία / Είδος                               | Σημασία (απλά)                                                                                                                                |
| :---------------------------------------------------- | :---------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------- |
| **Βασική Εισοδηματική Στήριξη για τη Βιωσιμότητα (BISS)** | Άμεση Ενίσχυση                                 | Η κύρια ενίσχυση που λαμβάνει ο παραγωγός βάσει των δικαιωμάτων του ("δικαιώματα").                                                              |
| **Αναδιανεμητική Ενίσχυση (πρώην Πρασίνισμα)**          | Άμεση Ενίσχυση                                 | Πρόσθετη ενίσχυση για παραγωγούς με συγκεκριμένο εύρος στρεμμάτων (20-110 στρ.).                                                                |
| **Εθνικό Απόθεμα**                                    | Άμεση Ενίσχυση                                 | Χορήγηση δικαιωμάτων βασικής ενίσχυσης σε νέους γεωργούς ή γεωργούς που ξεκινούν γεωργική δραστηριότητα.                                        |
| **Συμπληρωματική Εισοδηματική Στήριξη Νεαρών Γεωργών** | Άμεση Ενίσχυση                                 | Πρόσθετη ενίσχυση για παραγωγούς ηλικίας έως 40 ετών.                                                                                           |
| **Συνδεδεμένες Ενισχύσεις**                          | Άμεση Ενίσχυση                                 | Ενισχύσεις που συνδέονται με την παραγωγή συγκεκριμένων προϊόντων (π.χ. βαμβάκι, ρύζι, όσπρια) και απαιτούν τήρηση προϋποθέσεων.                |
| **Τομεακές Παρεμβάσεις Αγροτικής Ανάπτυξης (ΠΑΑ)**    | Προγράμματα Αγροτικής Ανάπτυξης (Δεύτερος Πυλώνας) | Προγράμματα όπως η Βιολογική Γεωργία, Εξισωτική Αποζημίωση, Απονιτροποίηση.                                                                     |
| **Εξισωτική Αποζημίωση**                             | Πρόγραμμα ΠΑΑ                                  | Ενίσχυση για παραγωγούς σε ορεινές, μειονεκτικές ή με ειδικά φυσικά μειονεκτήματα περιοχές.                                                     |
| **Απονιτροποίηση**                                    | Πρόγραμμα ΠΑΑ                                  | Πρόγραμμα μείωσης της νιτρορύπανσης γεωργικής προέλευσης.                                                                                       |
| **Δάσωση Γεωργικών Γαιών**                           | Πρόγραμμα ΠΑΑ                                  | Ενίσχυση για τη μετατροπή γεωργικής γης σε δασική.                                                                                             |
| **Οικολογικά Σχήματα (Eco-schemes)**                  | Άμεση Ενίσχυση (Πρώτος Πυλώνας)                 | Εθελοντικά ετήσια προγράμματα για την εφαρμογή φιλικών προς το περιβάλλον πρακτικών, με επιπλέον ενίσχυση (π.χ. λιπάσματα, κομποστοποίηση).         |
| **Βιολογική Γεωργία/Κτηνοτροφία (Πρόγραμμα ΠΑΑ)**       | Πρόγραμμα ΠΑΑ                                  | Πολυετές πρόγραμμα ενίσχυσης για τη μετατροπή ή διατήρηση σε βιολογικές μεθόδους παραγωγής.                                                     |
| **Βιολογική Γεωργία (Οικολογικό Σχήμα)**             | Οικολογικό Σχήμα                               | Ετήσια συμμετοχή σε πρακτικές βιολογικής γεωργίας.                                                                                             |

**3. Καλλιέργειες, Εκτάσεις & Σχετικοί Όροι**

| Τίτλος                                      | Κατηγορία / Είδος                | Σημασία (απλά)                                                                                                                                  |
| :------------------------------------------- | :------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------- |
| **Στρέμμα**                                  | Μονάδα Μέτρησης Έκτασης          | Βασική μονάδα μέτρησης της γεωργικής γης στην Ελλάδα (1 στρέμμα = 1000 τετραγωνικά μέτρα).                                                       |
| **Εκτάριο**                                  | Μονάδα Μέτρησης Έκτασης          | Μονάδα μέτρησης που χρησιμοποιείται συχνά στις αιτήσεις (1 εκτάριο = 10 στρέμματα).                                                              |
| **Ποικιλομορφία Καλλιεργειών**              | Υποχρέωση Παραγωγού             | Η υποχρέωση των παραγωγών (ανάλογα με τα στρέμματά τους) να καλλιεργούν περισσότερα από ένα είδη φυτών.                                           |
| **Αγρανάπαυση**                               | Είδος "Καλλιέργειας"            | Η γη που αφήνεται χωρίς καλλιέργεια για μια περίοδο, ώστε να ανακτήσει τη γονιμότητά της. Μπορεί να μετρήσει στην ποικιλομορφία.                   |
| **Πιστοποιημένος Σπόρος**                     | Είδος Σπόρου                    | Σπόρος που έχει παραχθεί και ελεγχθεί σύμφωνα με συγκεκριμένες προδιαγραφές, εξασφαλίζοντας την ποιότητα και την ταυτότητά του.                    |
| **Καρτελάκι Σπόρου**                         | Δικαιολογητικό                   | Η επίσημη ετικέτα που συνοδεύει τον πιστοποιημένο σπόρο και αναγράφει πληροφορίες όπως η ποικιλία, η σπορομερίδα κ.λπ.                            |
| **Σπορομερίδα**                              | Ταυτοποίηση Σπόρου              | Ένας μοναδικός κωδικός που αναγράφεται στο καρτελάκι και ταυτοποιεί μια συγκεκριμένη παρτίδα σπόρου.                                             |
| **Χειμερινά Σιτηρά**                         | Ομάδα Καλλιεργειών              | Καλλιέργειες που σπέρνονται το φθινόπωρο/χειμώνα και συγκομίζονται την άνοιξη/καλοκαίρι (π.χ. σιτάρι, κριθάρι).                                   |
| **Κτηνοτροφικά Φυτά**                        | Ομάδα Καλλιεργειών              | Καλλιέργειες που προορίζονται για ζωοτροφή (π.χ. τριφύλλι, βίκος, κτηνοτροφικός αραβόσιτος).                                                        |
| **Όσπρια**                                   | Ομάδα Καλλιεργειών              | Καλλιέργειες όπως φακές, ρεβίθια, φασόλια, μπιζέλια.                                                                                            |
| **Αραβόσιτος (Απλός)**                       | Είδος Καλλιέργειας              | Αραβόσιτος που καλλιεργείται για τον καρπό του.                                                                                                   |
| **Αραβόσιτος για Ενσίρωμα**                  | Είδος Καλλιέργειας              | Αραβόσιτος που συγκομίζεται ολόκληρο το φυτό για να γίνει ενσίρωμα (ζωοτροφή σε μπάλες).                                                          |
| **Ηλίανθος Ενεργειακός**                      | Είδος Καλλιέργειας              | Ηλίανθος που καλλιεργείται για παραγωγή βιοκαυσίμων (απαιτεί σύμβαση).                                                                           |
| **Ηλίανθος Ελαιούχος**                       | Είδος Καλλιέργειας              | Ηλίανθος που καλλιεργείται για παραγωγή λαδιού ή για άλλες χρήσεις (δεν απαιτεί υποχρεωτικά σύμβαση για συνδεδεμένη).                             |
| **Κηπευτικά**                                | Ομάδα Καλλιεργειών              | Διάφορα λαχανικά (π.χ. ντομάτες, αγγούρια, κρεμμύδια).                                                                                           |
| **Κηπευτικά Υποκάλυψη**                      | Είδος Καλλιέργειας              | Κηπευτικά που καλλιεργούνται σε θερμοκήπια.                                                                                                      |
| **Πατάτα (Γεώμηλα)**                         | Είδος Καλλιέργειας              | Η καλλιέργεια της πατάτας.                                                                                                                      |
| **Άρδευση (Αρδευόμενο/Ξηρικό)**              | Τρόπος Καλλιέργειας             | Δηλώνει αν μια καλλιέργεια ποτίζεται (αρδευόμενη) ή όχι (ξηρική).                                                                                 |
| **Χρήση Νερού (ΤΟΕΒ, Γεώτρηση)**             | Πηγή Άρδευσης                   | Η πηγή από την οποία προέρχεται το νερό για πότισμα (π.χ. Τοπικός Οργανισμός Εγγείων Βελτιώσεων, ιδιωτική γεώτρηση).                             |
| **Αγροτεμάχιο**                              | Μονάδα Γης                      | Ένα συνεχόμενο κομμάτι γης που δηλώνεται στην αίτηση.                                                                                           |
| **Χαρτογραφικό Υπόβαθρο (13ψήφιος)**         | Ταυτοποίηση Αγροτεμαχίου        | Μοναδικός κωδικός που αντιστοιχεί σε κάθε αγροτεμάχιο στο χάρτη του ΟΣΔΕ.                                                                       |
| **Περιφέρεια Δικαιωμάτων**                   | Κατηγορία Δικαιωμάτων           | Τα δικαιώματα βασικής ενίσχυσης κατηγοριοποιούνται σε περιφέρειες (Αρόσιμα, Μόνιμες Καλλιέργειες, Βοσκότοποι).                                    |

**4. Ιδιοκτησιακό Καθεστώς & Δικαιολογητικά**

| Τίτλος                                                | Κατηγορία / Είδος             | Σημασία (απλά)                                                                                                                             |
| :---------------------------------------------------- | :---------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| **Ιδιοκτησία (Αγροτεμαχίου)**                        | Νομικό Καθεστώς               | Ποιος έχει τη νόμιμη κατοχή ή χρήση του αγροτεμαχίου.                                                                                        |
| **Ιδιόκτητο**                                        | Είδος Ιδιοκτησίας            | Το αγροτεμάχιο ανήκει στον παραγωγό που υποβάλλει την αίτηση.                                                                                  |
| **Ε9**                                                | Φορολογικό Έντυπο             | Η δήλωση ακίνητης περιουσίας στην εφορία, όπου φαίνονται τα ιδιόκτητα αγροτεμάχια.                                                             |
| **ΑΤΑΚ (Αριθμός Ταυτότητας Ακινήτου)**                 | Ταυτοποίηση Ακινήτου          | Μοναδικός αριθμός που δίνεται σε κάθε ακίνητο στο Ε9.                                                                                         |
| **Ενοικιαζόμενο**                                     | Είδος Ιδιοκτησίας            | Ο παραγωγός νοικιάζει το αγροτεμάχιο από άλλον ιδιοκτήτη.                                                                                       |
| **Ιδιωτικό Συμφωνητικό Μίσθωσης (Ενοικιαστήριο)**      | Δικαιολογητικό                | Το έγγραφο που αποδεικνύει την ενοικίαση του αγροτεμαχίου.                                                                                     |
| **Μικτό (Ιδιοκτησιακό Καθεστώς)**                     | Είδος Ιδιοκτησίας            | Ένα μέρος του αγροτεμαχίου είναι ιδιόκτητο και ένα άλλο μέρος είναι ενοικιαζόμενο.                                                              |
| **Τιμολόγιο Αγοράς (γενικά)**                         | Δικαιολογητικό                | Απόδειξη αγοράς για διάφορα αγροτικά εφόδια (σπόροι, λιπάσματα, μηχανήματα κ.λπ.).                                                               |
| **Τιμολόγιο Πώλησης (γενικά)**                         | Δικαιολογητικό                | Απόδειξη πώλησης των αγροτικών προϊόντων.                                                                                                     |
| **Δελτίο Αποστολής**                                  | Δικαιολογητικό                | Έγγραφο που συνοδεύει τη μεταφορά εμπορευμάτων (π.χ. σπόρων πριν την έκδοση τιμολογίου).                                                        |
| **Σύμβαση (π.χ. Ηλίανθου, Ελαιοκράμβης)**              | Δικαιολογητικό                | Έγγραφο συμφωνίας μεταξύ παραγωγού και εταιρείας για την καλλιέργεια και παράδοση προϊόντος (π.χ. για βιοκαύσιμα).                              |
| **Εξουσιοδότηση (π.χ. ΕΛΓΑ)**                         | Δικαιολογητικό                | Έγγραφο με το οποίο ο παραγωγός δίνει την άδεια σε έναν φορέα (π.χ. ΕΛΓΑ) να προβεί σε μια ενέργεια για λογαριασμό του.                          |
| **Βεβαίωση Κτηνιάτρου (για Σιτηρέσιο)**               | Δικαιολογητικό                | Βεβαίωση από κτηνίατρο σχετικά με τις πρακτικές διαχείρισης ζωοτροφών για το οικολογικό σχήμα του σιτηρεσίου.                                  |
| **Μητρώο Εκμετάλλευσης (Κτηνοτρόφων)**                | Δικαιολογητικό                | Επίσημο βιβλίο όπου καταγράφονται τα ζώα μιας κτηνοτροφικής εκμετάλλευσης και οι μεταβολές τους (γεννήσεις, θάνατοι, σφαγές).                  |
| **Τιμολόγιο Σφαγής**                                 | Δικαιολογητικό                | Απόδειξη από το σφαγείο για τη σφαγή ζώων, απαραίτητη για ορισμένες συνδεδεμένες ενισχύσεις.                                                    |

**5. Οργανισμοί & Συστήματα**

| Τίτλος                                         | Κατηγορία / Είδος      | Σημασία (απλά)                                                                                                                             |
| :--------------------------------------------- | :--------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| **ΕΛΓΑ (Οργανισμός Ελληνικών Γεωργικών Ασφαλίσεων)** | Ασφαλιστικός Φορέας   | Ο κρατικός φορέας που ασφαλίζει τη φυτική παραγωγή και το ζωικό κεφάλαιο από φυσικούς κινδύνους.                                              |
| **ΟΓΑ (Οργανισμός Γεωργικών Ασφαλίσεων)**         | Ασφαλιστικός Φορέας   | Ο κύριος ασφαλιστικός φορέας (υγεία, σύνταξη) των αγροτών.                                                                                  |
| **ΔΟΥ (Δημόσια Οικονομική Υπηρεσία)**             | Φορολογική Αρχή      | Η εφορία.                                                                                                                                  |
| **Δασαρχείο**                                   | Δημόσια Υπηρεσία     | Αρμόδια υπηρεσία για θέματα δασών και δασικών εκτάσεων.                                                                                     |
| **ΔΟΒ (Διεπαγγελματική Οργάνωση Βάμβακος)**       | Οργάνωση Παραγωγών   | Οργάνωση που εκπροσωπεί τον τομέα του βαμβακιού. Η συμμετοχή είναι υποχρεωτική για τη συνδεδεμένη βάμβακος.                                    |
| **Συνεταιρισμός**                               | Οργάνωση Παραγωγών   | Ένωση παραγωγών με κοινούς σκοπούς (π.χ. προμήθεια εφοδίων, πώληση προϊόντων).                                                                |
| **Ομάδα Παραγωγών**                             | Οργάνωση Παραγωγών   | Επίσημα αναγνωρισμένη ομάδα παραγωγών που συνεργάζονται για την παραγωγή και εμπορία συγκεκριμένων προϊόντων.                                  |
| **ΑΑΔΕ (Ανεξάρτητη Αρχή Δημοσίων Εσόδων)**        | Δημόσια Αρχή         | Η αρχή που είναι υπεύθυνη για τη συλλογή φόρων και τη διαχείριση των φορολογικών δεδομένων (π.χ. Ε9, ΑΤΑΚ).                                       |

**6. Ζωικό Κεφάλαιο**

| Τίτλος                                    | Κατηγορία / Είδος                | Σημασία (απλά)                                                                                                              |
| :---------------------------------------- | :------------------------------- | :-------------------------------------------------------------------------------------------------------------------------- |
| **Ζωικό Κεφάλαιο**                          | Είδος Εκμετάλλευσης             | Τα ζώα που εκτρέφει ένας παραγωγός (π.χ. αγελάδες, πρόβατα, αίγες).                                                            |
| **Βοοειδή**                                 | Είδος Ζώων                      | Αγελάδες, ταύροι, μοσχάρια.                                                                                                  |
| **Αιγοπρόβατα**                             | Είδος Ζώων                      | Γίδες και πρόβατα.                                                                                                          |
| **Σφάγια**                                  | Ζώα προς Σφαγή                 | Ζώα που προορίζονται για σφαγή για την παραγωγή κρέατος.                                                                       |
| **Κατανομή Κοινοτικών Βοσκοτόπων**          | Ενίσχυση/Δικαίωμα              | Δικαίωμα χρήσης δημόσιων εκτάσεων βοσκής για τους κτηνοτρόφους, συχνά συνδεδεμένο με ενισχύσεις.                                   |
| **Σιτηρέσιο**                               | Διατροφή Ζώων                   | Το σύνολο των τροφών που καταναλώνουν τα ζώα. Σχετίζεται με οικολογικό σχήμα.                                                    |
| **Ιδιοκατανάλωση (Ζωοτροφών)**             | Χρήση Παραγωγής                 | Όταν ο παραγωγός χρησιμοποιεί τις ζωοτροφές που παράγει ο ίδιος για τα δικά του ζώα, αντί να τις πουλήσει.                         |
